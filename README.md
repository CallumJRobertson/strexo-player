# StrexoPlayer

**Stremio, streamlined.**

StrexoPlayer is a lightweight external player built to improve the Stremio experience across iOS and Android TV.

It focuses on faster playback, better reliability, and features that aren’t available in the default player — while keeping everything simple and familiar.

---

## ✨ Features

- 🚀 Fast, reliable playback
- 🎬 Improved stream handling (debrid, direct, etc.)
- ⏭️ Intro skipping support
- 🔊 Advanced audio controls (including delay adjustments)
- 📺 Android TV optimised UI + controls
- 🍏 Native iOS playback with better format support
- 🔄 Stremio progress syncing
- 🎯 Ongoing improvements based on community feedback

---

## 📱 Platforms

- iOS (TestFlight)
- Android TV (APK)
